Course homework assignments for "Fundamentals of Website Development" from Harvard Summer School 2015.

## Overview

This repository contains homework assignments completed during a web development course at Harvard University in Summer 2015. The assignments demonstrate foundational web development concepts including HTML structure, CSS styling, and basic PHP server-side scripting.

**Note**: This code is provided for review, study, and research purposes only.

## Course Information

- **Course**: Fundamentals of Website Development
- **Institution**: Harvard University (Summer School)
- **Year**: 2015
- **Focus**: HTML, CSS, JavaScript, PHP basics

## Prerequisites

- Web browser (Chrome, Firefox, Safari, etc.)
- PHP environment (for PHP files)
- Text editor or IDE

## Running the Code

### HTML/CSS Files

```bash
# Simply open HTML files in your browser
open index.html
```

### PHP Files

```bash
# Option 1: Use PHP built-in server
php -S localhost:8000

# Then visit http://localhost:8000 in your browser

# Option 2: Use Apache/Nginx with PHP installed
# Place files in web server document root
```
